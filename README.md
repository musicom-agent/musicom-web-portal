# Musicom Web Portal

Composer-facing frontend for interacting with the Musicom Framework.

## Tech Stack
- **Framework**: Next.js / React
- **Score Rendering**: Flat.io Embed API / VexFlow
- **Dashboard**: High-contrast Metrical Gravity Visualization (Tailwind CSS)

## Key Features
1. **DNA Editor**: Grid UI for Euclidean rhythms and Pitch matrices.
2. **Real-time Theory Guard**: Highlights "Illegal" counterpoint/harmony notes *before* rendering.
3. **Project Portfolio**: Link to GitHub Pages dashboard index.
4. **Library Browser**: Access to `Styles/` knowledge base.

## Development
```bash
npm install
npm run dev
```
